DS Query Tutorial

Begin by importing the DirectoryService class from the directory module.

from pancloud.directory import DirectoryService

Next, define a DirectoryService instance and point it to a Directory-Sync Service API.

url = 'https://app-stg1.directorysync.paloaltonetworks.com'
ds = DirectoryService(
    url,
    cert='ssl/ds.client.pem',
    verify=False
)

Prepare the query data payload.

data = {
    "tenantId": "5748120208644591984",
    "domain": "ad.daniel.com"
}

Generate a new query.

q = ds.cache(data, object="user")
