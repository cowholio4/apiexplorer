PAN Cloud SDK

PAN Cloud Python SDK Python idiomatic SDK for the Palo Alto Networks Application Framework. The PAN Cloud Python SDK was created to assist 3rd-party developers in their interactions with the various APIs implemented by the Palo Alto Networks Application Framework. The primary goal is to provide full, low-level API coverage of the following: Logging Service API Directory Service API Event Service API After understanding all the various ways applications will need to consume these APIs, the secondary goal is to provide high-level coverage, in the form of helper methods, for common tasks/operations. Overview: https://github.com/PaloAltoNetworks/pancloud Documentation: https://pancloud.readthedocs.io Free software: ISC license

Begin by importing the DirectoryService class from the directory module.

from pancloud.directory import DirectoryService

Next, define a DirectoryService instance and point it to a Directory-Sync Service API.

url = 'https://app-stg1.directorysync.paloaltonetworks.com'
ds = DirectoryService(
    url,
    cert='ssl/ds.client.pem',
    verify=False
)

Prepare the query data payload.

data = {
    "tenantId": "5748120208644591984",
    "domain": "ad.daniel.com"
}

Generate a new query.

q = ds.cache(data, object="user")
